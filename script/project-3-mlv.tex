\textbf{Research question} is there a difference between the chicks
weights across the diet groups?

\section{Question 1}\label{question-1}

\subsection{Formulate a one-way ANOVA model for the
problem}\label{formulate-a-one-way-anova-model-for-the-problem}

The one-way ANOVA model is:

\[
Y_{ij} = \mu + \alpha_j + \epsilon_{ij} \quad \text{for} \quad i = 1, \ldots, 71; \quad j = 1, \ldots, 6; \quad \epsilon_{ij} \overset{\text{iid}}{\sim} N(0, \sigma^2)
\]

where ( Y\_\{ij\} ) is the ( i )-th weight from the ( j )-th feed, and (
\alpha\_j ) is the effect for the ( j )-th feed group.

\subsection{Formulate the null hypothesis and the
alternative}\label{formulate-the-null-hypothesis-and-the-alternative}

The hypotheses associated with the model are:

\[
H_0 : \alpha_1 = \alpha_2 = \cdots = \alpha_6 = 0
\]

\[
H_1 : \text{at least one } \alpha_i \text{ is different from } 0
\] \#\# Formulate the test statistic

The test statistic is given as:

\[
F = \frac{\sum_{j=1}^6 n_j (\bar{x}_j - \bar{x})^2 / 5}{\sum_{i=1}^{n_j} \sum_{j=1}^6 (x_{ij} - \bar{x}_j)^2 / 65}
\] and the critical value is found in a table of probability values for
the F distribution with (degrees of freedom) ( \text{df}\_1 = 5,
\text{df}\_2 = 65. ) Also, ( n\_j ) = the sample size in the ( j )-th
group, ( \bar\{x\}\_j ) is the sample mean of the ( j )-th group, and (
\bar\{x\} ) is the overall mean.
